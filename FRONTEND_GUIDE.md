# 前端系統使用說明書

> 適用對象：廚房現場備料人員
> 版本：2026-04

---

## 開啟方式

用 **Safari** 開啟系統網址，點擊底部分享按鈕 → 「加入主畫面」，即可像 App 一樣使用。

> ⚠️ 請勿使用無痕模式，資料會在關閉後消失。

---

## 頁面一覽

| 分頁 | 用途 |
| :--- | :--- |
| 叫貨單 | 填寫今日向供應商叫貨的數量 |
| 匯出 | 複製叫貨單 / 損耗紀錄，貼到 LINE |
| 損耗回報 | 紀錄食材備料過程中的重量變化 |
| 設定 | 新增品項、修改縮寫 |

---

## 叫貨單

### 操作步驟

1. 在每個品項右側欄位輸入今天要叫的**數量**
2. 不需要叫的品項留空或填 `0`
3. 確認後點「**✓ 確認送出**」，系統自動跳到匯出頁面

### 注意事項

- 輸入純數字，單位由系統自動帶入（例：`5`，系統顯示 `5kg`）
- 點「**✕ 清空**」可一鍵清除所有數量

---

## 損耗回報

### 操作步驟

1. 填寫**日期**（8 位數，例：`20260411`）與**負責人**姓名
2. 找到對應品項，依序填入三欄重量：

| 欄位 | 說明 |
| :--- | :--- |
| 進貨(g) | 原始採購回來的重量 |
| 退冰(g) | 退冰後秤重（冷凍食材才需填，蔬果留空） |
| 成品(g) | 處理完畢、可出餐的重量 |

3. 點「**✓ 確認送出**」，自動跳到匯出頁面

### 重量填寫規則

三欄數值必須符合以下大小關係，否則系統會**拒絕送出**並顯示錯誤：

```
進貨重量  ≥  退冰重量  ≥  成品重量
```

**範例（正確）**

| 品項 | 進貨 | 退冰 | 成品 |
| :--- | :---: | :---: | :---: |
| 牛小排 | 10000 | 9600 | 9000 |
| 節瓜 | 3000 | （留空） | 2000 |

**範例（錯誤，系統會擋下）**

| 狀況 | 原因 |
| :--- | :--- |
| 退冰填 `10500`，進貨填 `10000` | 退冰後不可能比進貨重 |
| 成品填 `3500`，進貨填 `3000` | 成品不可能比進貨重 |

### 錯誤訊息說明

| 訊息 | 修正方法 |
| :--- | :--- |
| 請填寫8位數日期 | 確認日期格式為 `YYYYMMDD`，共8碼 |
| 請填寫負責人 | 填入姓名後再送出 |
| 數據邏輯錯誤，重量必須符合「進貨 ≥ 退冰 ≥ 成品」 | 重新核對三欄數值 |
| 數據邏輯錯誤，成品重量不可大於進貨重量 | 成品重量填寫有誤，重新秤重確認 |
| 請至少填寫一項品項的進貨與成品重量 | 至少要填一個品項才能送出 |

## 匯出

- **叫貨單**：點「📋 複製」後直接貼到 LINE
- **損耗紀錄**：若筆數超過 10 筆，系統會自動分成多批，請按順序逐批複製貼上

---

## 設定

### 修改縮寫

在品項右側欄位修改縮寫，點「💾 儲存」生效。

### 新增品項

填寫品項名稱、類別、單位、縮寫後點「＋ 新增」。

> ⚠️ 新增的品項名稱必須與 **Google 試算表白名單**一致，否則 LINE 機器人會拒絕寫入。

### 刪除損耗紀錄

前往「設定 → 資料管理」，點「清除」可刪除所有本機損耗紀錄。
